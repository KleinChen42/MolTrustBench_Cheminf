# Public scope and interpretation boundary

## Included

This compact package contains reusable chemistry and public-release-index code, five derived CSV summaries, three vector figures, and a deterministic checksum/invariant verifier. The case-study summaries are restricted to declared chemical relations and release-anchor sensitivity.

## Excluded

The package does not distribute raw ChEMBL archives, raw benchmark records, outcome values, labels, predictions, model-training artifacts, credentials, caches, controlled source forms, or manuscript-only supporting material.

## Permitted interpretation

The included outputs describe public-source observability under the stated molecular transformation, identity-exclusion, source-panel, and release-anchor contracts.

## Not inferred

No file in this repository supports a claim that a model was trained on a particular molecule, memorized a molecule, leaked information, or experienced a causal performance effect from public availability.
